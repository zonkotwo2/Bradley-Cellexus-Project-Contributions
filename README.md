# Bradley Cellexus Project Contributions

This repository showcases my individual contributions to a group biomedical engineering project in which a team of five developed a bioreactor concept for Cellexus. The purpose of this repository is to provide clear evidence of the technical work I completed across CAD design, electronics, and embedded control, rather than simply listing the project on my CV.

## Project Overview

The project involved designing and developing components of a bioreactor system for use in a biomedical engineering context. My work focused on practical engineering contributions across three areas:

- Mechanical design and prototyping
- Electronics design
- Embedded control logic for system operation

This repository is split into branches so that each area of my contribution can be viewed separately.

## My Contribution

My individual contribution to the project centred on supporting the design and development of functional hardware and control elements used within the wider bioreactor system. This included:

- Creating and editing CAD parts and drawings in SolidWorks
- Preparing files for prototyping and 3D printing
- Contributing to circuit design work for the electronic system
- Developing embedded Arduino-based control code for sensors and actuator behaviour
- Contributing to an integrated engineering workflow across mechanical, electronic, and software elements

This repository is intended to show the breadth of that contribution in a more transparent way than a single CV bullet point can.

## Repository Structure

### `main`

The `main` branch acts as the landing page for the repository and explains the purpose of the project and how the contributions are organised.

### `Solidworks-Contributions`

This branch contains my CAD and prototyping-related work, including SolidWorks part files, drawing files, STL exports, 3MF print files, and generated printer files.

Examples of files in this branch include:

- Impeller-related design files
- Luer-lock concept files
- Barb component files
- Motor mount and other printable parts
- STL, 3MF, and print output files used for prototyping

This branch demonstrates my involvement in design iteration, manufacturable part development, and prototype preparation.

### `Electronic-Contributions`

This branch contains my electronics design contribution in the form of circuit design files.

These files represent work related to the electrical side of the bioreactor system and show my involvement beyond purely mechanical design.

### `C++-Contributions`

This branch contains my embedded control code for the project.

The code in `FINAL_FINAL.ino` reflects work on integrating sensing and control behaviour within the system, including:

- Sensor reading and monitoring
- Control logic for operating hardware based on measured conditions
- Actuator behaviour for pump-related response
- Serial output for testing and debugging

This branch is included to evidence practical programming experience in a hardware-linked engineering project.
